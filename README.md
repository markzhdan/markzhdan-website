## Stack

- **Framework** - Next.js 16 (App Router)
- **Database** - MongoDB Atlas
- **Styling** - Tailwind CSS
- **Auth** - JWT (admin-only)
- **Deployment** - Vercel

## Pages

- `/` - About, projects, socials, resume
- `/experience` - Work history
- `/projects` - Portfolio of projects
- `/blogs` - Articles and daily log
